Playing with rasm2

Radare2’s assembler/disassembler is rasm2, and albeit being used internally, it is also a standalone binary that you can use. It can of course disassemble $ rasm2 -d 89d85d90 mov eax, ebx;pop ebp;nop but also assemble $ rasm2 'mov eax, ebx;pop ebp;nop' 89d85d90 Not only x86, but also mips $ rasm2 -a mips 'addiu a1, a2, 8' 0800c524 $ rasm2 -a mips -d 0800c524 addiu a1, a2, 8 and many more.

YARA support

We now have (experimental) YARA support inside radare2. If you are building from the latest git, you just have to install libyara, no need to recompile anything. ASCII graphs!

We may not have a GUI like IDA, but we still have some graphs. This is a small (200 lines of code) proof of concept, but there is more to come colors utf-8 layouts resizing animations … You can try this new feature with VV if you are using radare2 from git. And by the way, this is documented, and has some tests to avoid regressions. Feel free to take a look at the /libr/cons folder if you want to contribute.
